<h1>Viking UI</h1>

`viking` 是一个模仿 Ant Deisgn 的 React UI 组件库


## ✨ 特性

- TypeScript 编写
- Storybook 组件调试和文档生成

## 📦 安装

```bash
# With npm
npm install viking

# With yarn
yarn add viking
```

## 🔨 使用
```js
import { Button } from 'viking'

ReactDOM.render(<Button>Hello, viking</Button>, root)
```
