# notaui
`notaui` UI Library Demo

## ✨ 特性

- TypeScript 编写
- SCSS 样式控制
- Storybook 组件文档生成
- 单元测试覆盖

## 📦 安装

```bash
# With npm
npm install notaui

# With yarn
yarn add notaui
```

## 🔨 使用
```js
import { Button } from 'notaui'

ReactDOM.render(<Button>Hello, notaui</Button>, root)
```
