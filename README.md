# Viking UI
`viking` 是一个模仿 Ant Deisgn 的 React UI 组件库, 代码来源于慕课网


## ✨ 特性

- TypeScript 编写
- SCSS 样式
- Storybook 组件文档生成

## 📦 安装

```bash
# With npm
npm install viking

# With yarn
yarn add viking
```

## 🔨 使用
```js
import { Button } from 'viking'

ReactDOM.render(<Button>Hello, viking</Button>, root)
```
